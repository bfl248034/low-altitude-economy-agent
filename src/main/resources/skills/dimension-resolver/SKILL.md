# 维度解析技能 (Dimension Resolver)

## 功能
从用户自然语言查询中解析时间、地区、学历等维度条件，并转换为标准化编码。

## 支持的维度

### 1. 时间维度
支持多种时间表达方式：

| 自然语言 | 解析结果 | 说明 |
|---------|---------|------|
| "近6个月" | time_range: 最近6个月 | 动态计算 |
| "2024年1月" | time_id: 202401 | 标准化编码 |
| "最近3个月" | time_range: 最近3个月 | 动态计算 |
| "2024年" | time_range: 2024全年 | 年度范围 |

**相对时间推算规则**：
- "最近N个月" → 从当前月往前推N个月
- "近N个月" → 同"最近N个月"
- "本月" → 当前月份
- "上月" → 上个月份

### 2. 地区维度
支持地区名称和同义词匹配：

| 地区编码 | 标准名称 | 同义词 |
|---------|---------|--------|
| 110000 | 北京 | 北京市、京城、首都 |
| 310000 | 上海 | 上海市、魔都 |
| 440100 | 广州 | 广州市、羊城 |
| 440300 | 深圳 | 深圳市、鹏城 |
| 330100 | 杭州 | 杭州市 |

**地区级别识别**：
- "各省"、"各省份" → region_level = 2（省级）
- "各市"、"各地市" → region_level = 3（地市级）
- 具体城市名 → region_id = 对应编码

### 3. 学历维度

| 学历编码 | 标准名称 | 同义词 |
|---------|---------|--------|
| 0 | 不限 | 不限学历 |
| 1 | 高中/中专/中技 | 高中、中专 |
| 2 | 大专 | 专科、大专 |
| 3 | 本科 | 本科、学士 |
| 4 | 硕士 | 硕士研究生 |
| 5 | 博士 | 博士研究生 |

**多值支持**：
- "本科和硕士" → edu_level IN ('3', '4')
- "本科及以上" → edu_level >= '3'

## 输出格式

```json
{
  "dimensions": {
    "time": {
      "type": "range",
      "start": "202401",
      "end": "202406",
      "description": "近6个月"
    },
    "region": {
      "type": "specific",
      "regionId": "110000",
      "regionName": "北京",
      "level": null
    },
    "eduLevel": {
      "type": "specific",
      "values": ["3"],
      "names": ["本科"]
    }
  },
  "analysisType": "trend",
  "reasoning": "解析逻辑说明"
}
```

## 分析类型识别

| 关键词 | 分析类型 | 说明 |
|-------|---------|------|
| "排名"、"排行" | ranking | 按值排序 |
| "趋势"、"走势" | trend | 时间序列变化 |
| "对比"、"比较"、"不同" | comparison | 多维度对比 |
| "分布"、"占比" | distribution | 占比分析 |

## 示例

**用户输入**: "北京近6个月本科招聘薪资"

**解析结果**:
```json
{
  "dimensions": {
    "time": {"type": "range", "start": "202401", "end": "202406"},
    "region": {"type": "specific", "regionId": "110000", "regionName": "北京"},
    "eduLevel": {"type": "specific", "values": ["3"], "names": ["本科"]}
  },
  "analysisType": "trend",
  "reasoning": "解析出具体地区北京、时间范围近6个月、学历本科"
}
```

**用户输入**: "各省份招聘薪资排名"

**解析结果**:
```json
{
  "dimensions": {
    "time": {"type": "latest", "count": 1},
    "region": {"type": "level", "level": 2, "name": "省级"},
    "eduLevel": null
  },
  "analysisType": "ranking",
  "reasoning": "按省级地区分组，取最新月份数据做排名"
}
```
