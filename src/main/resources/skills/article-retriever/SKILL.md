# 文章检索技能 (Article Retriever)

## 状态
🚧 **开发中**

## 规划功能

### 1. 文章索引
- 低空经济新闻报道
- 行业分析文章
- 企业动态资讯
- 技术论文

### 2. 检索方式
- 语义检索（向量相似度）
- 关键词检索
- 时间范围过滤
- 来源筛选

### 3. 输出格式
- 文章摘要
- 关键信息提取
- 相关文章推荐

## 预期使用

```
用户：最近有什么低空经济的新闻？

系统：
[调用 article-retriever Skill]

回复：
为您找到以下近期低空经济新闻：

1. 《XX公司完成亿元融资》...（来源：XX网，2天前）
2. 《低空经济政策出台》...（来源：XX报，1周前）
...
```

## 技术方案
- 向量数据库：Elasticsearch / Milvus
- Embedding：DashScope 文本向量
- Rerank：LLM 精排
