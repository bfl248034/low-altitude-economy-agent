# 意图路由技能 (Intent Router)

## 功能
分析用户查询，判断意图类型，决定后续处理流程。

## 输入
```json
{
  "userQuery": "北京招聘薪资是多少？"
}
```

## 输出
```json
{
  "intent": "DATA_QUERY",
  "confidence": 0.95,
  "keywords": ["招聘", "薪资", "北京"],
  "reasoning": "用户查询包含数据指标关键词"
}
```

## 意图类型

| 意图 | 关键词 | 处理方式 |
|------|--------|---------|
| CHAT | 你好、您好、嗨、在吗、谢谢、再见、帮助、介绍 | chat_agent |
| DATA_QUERY | 招聘、薪资、工资、岗位、企业、专利、数量、排名、趋势 | data_query_agent |
| ARTICLE | 文章、新闻、资讯、报道 | 回复开发中 |
| POLICY | 政策、法规、文件、规定 | 回复开发中 |

## 决策规则

### 1. 闲聊识别
- 问候语："你好"、"您好"、"嗨"、"在吗"
- 感谢/告别："谢谢"、"感谢"、"再见"、"拜拜"
- 帮助请求："能做什么"、"有什么功能"、"帮助"

### 2. 数据查询识别
**指标关键词**：
- 招聘类：招聘、招工、岗位、职位、用工、人才
- 薪资类：薪资、工资、薪酬、收入、待遇、报酬
- 企业类：企业、公司、主体、注册
- 创新类：专利、发明、实用新型

**分析类型关键词**：
- 趋势：趋势、走势、变化、增长
- 排名：排名、排行、第几、前几
- 对比：对比、比较、vs、 versus
- 分布：分布、占比、结构

**地区关键词**：
- 具体地区：北京、上海、深圳、广州、杭州
- 级别描述：全国、各省、各城市、各区

**时间关键词**：
- 相对时间：近X个月、最近X年
- 绝对时间：2024年、去年、今年、上月

### 3. 文章检索识别
- 文章、新闻、资讯、报道、媒体

### 4. 政策分析识别
- 政策、法规、规定、文件、条例、意见

## 完整示例

**输入**："北京近6个月本科招聘薪资"

**处理**：
1. 非闲聊
2. 包含"招聘"、"薪资" → DATA_QUERY
3. 包含"北京" → 地区
4. 包含"近6个月" → 时间
5. 包含"本科" → 学历维度

**输出**：
```json
{
  "intent": "DATA_QUERY",
  "confidence": 0.98,
  "keywords": ["招聘", "薪资", "北京", "6个月", "本科"],
  "reasoning": "数据查询：招聘薪资指标，北京地区，6个月趋势"
}
```
