# 意图路由技能 (Intent Router)

## 功能
快速识别用户输入的主要意图，将请求路由到合适的处理路径。

## 输入
- `userQuery`: string - 用户原始输入

## 输出
```json
{
  "intent": "CHAT | DATA_QUERY | UNKNOWN",
  "confidence": 0.95,
  "reasoning": "用户询问招聘薪资，包含指标关键词",
  "nextAction": "chat_response | data_query"
}
```

## 决策规则

### CHAT (闲聊)
**触发条件**（满足任一）：
- 问候语：你好、在吗、谢谢、再见、早上好、晚上好
- 帮助请求：help、帮助、能做什么、介绍
- 简单确认：好的、明白、知道了

**处理**：直接生成友好回复，无需调用其他Skill

### DATA_QUERY (数据查询)
**触发条件**（满足任一）：
- 指标关键词：招聘、薪资、工资、薪酬、企业、公司、数量、岗位、职位、专利、排名、趋势、同比、占比
- 地区关键词 + 指标：北京招聘、各省企业数
- 时间关键词 + 指标：近3个月薪资、2024年趋势

**处理**：调用 `data-query-orchestrator` Skill

### UNKNOWN (未知)
**触发条件**：
- 无法明确分类
- 语义模糊

**处理**：友好询问用户具体需求，提供示例

## 快速判断逻辑

```
1. 检查是否匹配CHAT关键词 → 是 → 返回CHAT
2. 检查是否包含指标/数据关键词 → 是 → 返回DATA_QUERY
3. 以上都不匹配 → 返回UNKNOWN
```

## 响应示例

| 用户输入 | 判定结果 | 下一步 |
|---------|---------|--------|
| "你好" | CHAT | 直接回复问候 |
| "北京招聘薪资多少" | DATA_QUERY | 调用data-query-orchestrator |
| "能做什么" | CHAT | 介绍能力 |
| "帮我查一下" | UNKNOWN | 询问查什么 |
