# 结果格式化技能 (Result Formatter)

## 功能
将数据库查询结果转换为人类可读的格式，包括编码翻译、数值格式化和展示建议。

## 编码翻译

### 地区编码翻译
| 编码 | 中文名 |
|------|--------|
| 110000 | 北京 |
| 310000 | 上海 |
| 440100 | 广州 |
| 440300 | 深圳 |
| 330100 | 杭州 |

### 学历编码翻译
| 编码 | 中文名 |
|------|--------|
| 0 | 不限 |
| 1 | 高中/中专/中技 |
| 2 | 大专 |
| 3 | 本科 |
| 4 | 硕士 |
| 5 | 博士 |

### 时间编码翻译
| 编码 | 中文名 |
|------|--------|
| 202401 | 2024年1月 |
| 202402 | 2024年2月 |
| ... | ... |

## 数值格式化

### 薪资格式化
- 原始值: 12500
- 格式化: "12,500元" 或 "1.25万元"

### 数量格式化
- 原始值: 1500000
- 格式化: "1,500,000个" 或 "150万个"

## 展示形式推荐

根据分析类型推荐合适的展示方式：

| 分析类型 | 推荐展示 | 说明 |
|---------|---------|------|
| trend | 折线图 | 时间序列趋势 |
| ranking | 柱状图/表格 | 排名对比 |
| comparison | 分组柱状图 | 多维度对比 |
| distribution | 饼图 | 占比分布 |
| single | 指标卡片 | 单一数值 |

## 输出格式

```json
{
  "formattedText": "格式化后的文本描述",
  "chartType": "line",
  "chartData": {
    "xAxis": ["2024-01", "2024-02", ...],
    "series": [{
      "name": "本科招聘薪资",
      "data": [12500, 12800, ...]
    }]
  },
  "summary": "数据总结"
}
```

## 示例

**输入**: 
- 指标: 招聘岗位平均薪酬
- 分析类型: trend
- 查询结果: [{"time_id": "202401", "fact_value": 12500}, {"time_id": "202402", "fact_value": 12800}]

**输出**:
```json
{
  "formattedText": "北京近6个月本科招聘薪资趋势：从2024年1月的12,500元上涨至2024年2月的12,800元...",
  "chartType": "line",
  "chartData": {
    "xAxis": ["2024-01", "2024-02"],
    "series": [{
      "name": "本科招聘薪资（元）",
      "data": [12500, 12800]
    }]
  },
  "summary": "整体呈上涨趋势，平均薪资约12,650元"
}
```
