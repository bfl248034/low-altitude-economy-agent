# 结果展示技能 (Result Presenter)

## 功能
将查询结果数据格式化为用户友好的文本展示。

## 输入
```json
{
  "indicators": [{"indicatorId": "...", "indicatorName": "招聘薪资"}],
  "dimensions": {
    "timeRange": {"start": "2024-01-01", "end": "2024-06-30"},
    "region": {"name": "北京"},
    "analysisType": "TREND"
  },
  "data": [...],
  "summary": "整体呈上涨趋势"
}
```

## 输出
用户友好的文本回复：
```
北京近6个月招聘薪资趋势：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
时间        数值（元）  环比
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2024年6月   12,800     +2.3%
2024年5月   12,500     +1.2%
...
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 分析：整体呈上涨趋势，平均薪资约12,650元

💡 相关问题：
• 上海同期招聘薪资对比
• 本科学历薪资趋势
• 不同城市招聘数量排名
```

## 展示类型

### 1. 趋势展示 (TREND)
- 时间序列数据表格
- 环比/同比变化
- 趋势线描述

### 2. 排名展示 (RANKING)
- 排名列表（TOP N）
- 数值对比
- 排名变化标识

### 3. 对比展示 (COMPARISON)
- 并列对比表格
- 差异高亮
- 相对百分比

### 4. 分布展示 (CROSS_SECTION)
- 占比表格
- 分布结构描述
- 集中度分析

### 5. 单值展示 (SINGLE)
- 核心数值突出
- 相关维度说明
- 参考对比

## 格式化规则

### 数值格式化
- ≥1亿：X.XX亿
- ≥1万：X.XX万
- <1万：千分位

### 单位处理
- 货币：元、万元、亿元
- 数量：个、家、项
- 百分比：%

### 时间格式化
- 月表：2024年6月
- 年表：2024年
- 日表：2024-06-30

## 使用 Tool

### ResultFormatTool

**formatNumber**: 格式化数值
```java
formatNumber(value=12500, unit="元") → "1.25万元"
```

**generateSummary**: 生成摘要
```java
generateSummary(data, analysisType="TREND", indicatorName="招聘薪资")
→ "整体呈上涨趋势，平均薪资约12,650元"
```

**suggestQueries**: 推荐问题
```java
suggestQueries(indicatorId, dimensionId)
→ ["上海同期对比", "本科学历趋势", "城市排名"]
```

## 完整示例

**输入数据**：
```json
{
  "indicators": [{"indicatorName": "招聘岗位平均薪酬", "unit": "元"}],
  "dimensions": {
    "region": {"name": "北京"},
    "analysisType": "TREND"
  },
  "data": [
    {"time_id": "202406", "fact_value": 12800},
    {"time_id": "202405", "fact_value": 12500},
    {"time_id": "202404", "fact_value": 12300}
  ]
}
```

**输出展示**：
```
🔍 北京招聘岗位平均薪酬趋势

📅 时间范围：2024年4月 - 2024年6月

┌──────────┬──────────┬────────┐
│ 时间      │ 平均薪酬  │ 环比    │
├──────────┼──────────┼────────┤
│ 2024年6月 │ 12,800元 │ +2.4%  │
│ 2024年5月 │ 12,500元 │ +1.6%  │
│ 2024年4月 │ 12,300元 │ -      │
└──────────┴──────────┴────────┘

📈 分析结论：
近3个月北京招聘薪资呈持续上涨趋势，
平均薪酬约12,533元，涨幅约4.1%。

💡 您可能还想了解：
• 上海招聘薪资同期对比
• 各学历薪资差异
• 不同城市招聘热度排名
```
