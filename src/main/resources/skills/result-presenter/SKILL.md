# 结果展示技能 (Result Presenter)

## 功能
接收数据查询的结果，生成美观、易读的最终回复。

## 输入
```json
{
  "success": true,
  "indicators": [...],
  "dimensions": {...},
  "data": [...],
  "summary": "数据摘要"
}
```

## 输出
```json
{
  "responseText": "北京近6个月本科招聘薪资趋势：\n...",
  "chartType": "line",
  "chartData": {...},
  "suggestedQueries": [
    "硕士学历薪资对比",
    "其他城市薪资趋势"
  ]
}
```

## 格式化规则

### 趋势数据 (TREND)
```
[地区][时间范围][维度]的[指标名称]趋势：

[时间点1]：[数值]
[时间点2]：[数值]
...

[数据摘要]
```

### 排名数据 (RANKING)
```
[地区级别][指标名称]排名（[时间]）：

1. [地区1]：[数值]
2. [地区2]：[数值]
...

[数据摘要]
```

### 对比数据 (COMPARISON)
```
[指标名称]对比分析：

[对比项1]：[数值]
[对比项2]：[数值]

差异：[差异描述]
```

### 截面数据 (CROSS_SECTION)
```
按[维度]分布的[指标名称]：

[维度值1]：[数值] ([占比])
[维度值2]：[数值] ([占比])
...
```

## 图表建议

| 分析类型 | 推荐图表 | 说明 |
|---------|---------|------|
| TREND | line | 折线图展示趋势 |
| RANKING | bar | 柱状图展示排名 |
| COMPARISON | groupedBar | 分组柱状图对比 |
| CROSS_SECTION | pie/bar | 饼图或柱状图分布 |

## 推荐问题生成

基于当前查询上下文，推荐相关问题：

**基于指标**：
- 该指标的其他维度分析
- 该指标的趋势变化
- 该指标的地区对比

**基于维度**：
- 切换维度值（如"硕士学历对比"）
- 扩展维度（如"加入工作经验维度"）

**基于地区**：
- 其他地区对比
- 该地区其他指标

## 示例

**输入**：
```json
{
  "indicators": [{"indicatorName": "招聘岗位平均薪酬", "unit": "元"}],
  "dimensions": {
    "region": {"name": "北京"},
    "timeRange": {"start": "2024-01", "end": "2024-06"},
    "analysisType": "TREND"
  },
  "data": [...],
  "summary": "整体呈上涨趋势，平均薪资约12,650元"
}
```

**输出**：
```

━━━━━━━━━━━━━━━━━━━━━

2024年2月  │  12,500元
```

## 错误回复

当查询失败时：
```
抱歉，暂时无法获取该数据。

可能原因：
• 该指标暂无数据
• 查询条件过于具体
• 系统暂时不可用

建议：
• 尝试更通用的查询，如"全国招聘薪资"
• 换个时间范围，如"近3个月"
• 稍后重试
```
