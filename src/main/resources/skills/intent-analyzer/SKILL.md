# 意图分析技能 (Intent Analyzer)

## 功能
识别用户输入的主要意图类型。

## 意图分类

### 1. CHAT (闲聊)
**关键词**：你好、在吗、谢谢、再见、早上好、help、帮助
**示例**：
- "你好"
- "能做什么"
- "谢谢"

### 2. DATA_QUERY (数据查询)
**关键词**：招聘、薪资、工资、企业、公司、数量、岗位、职位、专利、排名、趋势
**示例**：
- "北京招聘薪资"
- "各省份企业数量"
- "近3个月岗位趋势"

### 3. ARTICLE_QUERY (文章查询) [预留]
**关键词**：新闻、文章、动态、资讯
**示例**：
- "低空经济最新新闻"

### 4. POLICY_QUERY (政策查询) [预留]
**关键词**：政策、法规、文件、通知
**示例**：
- "低空经济相关政策"

### 5. UNKNOWN (未知)
无法明确分类的输入。

## 输出格式

```json
{
  "intent": "DATA_QUERY",
  "confidence": 0.95,
  "reasoning": "用户询问招聘薪资，属于数据查询"
}
```

## 处理规则
- 如识别为 CHAT，直接生成友好回复
- 如识别为 DATA_QUERY，继续调用 indicator-matcher
- 如识别为 ARTICLE_QUERY 或 POLICY_QUERY，提示功能开发中
- 如识别为 UNKNOWN，询问用户具体需求
